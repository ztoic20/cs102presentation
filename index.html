Data Collection Method

Data Collection Method: Manual web browsing from different websites, observational techniques, transactional data, surveys, and experimental data.

Each of us conducted independent research to locate credible sources of data on animal extinctions. After that, we used the IUCN Red List website as the main tool to verify whether the species exists and if it is truly an extinct species. This helped ensure that the gathered data from different sources is reliable and relevant.

Data Presentation

Using the information that each of us collected, we created a table to present data on the distribution and causes of animal extinctions per continent.

The table includes columns for the animal's common name, scientific name, date of extinction, continent, and primary reason for extinction.

Collaborative Data Cleaning

After creating a table, we converted it into a CSV file and named it extinct_species.csv. We opened Jupyter and uploaded the CSV file that we created. Then we created a new Python file in Jupyter.

After that, as a team, we worked together to clean the collected data by identifying and addressing any missing or duplicate information. We also made sure that the data is standardized to allow accurate analysis and comparison.
