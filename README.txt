Upload all files inside this zip to your htdocs folder.

Example:
htdocs/your-folder-name/
  index.html
  styles.css
  scripts.js
  extinct_animal.csv
  extinct_animal_cleaned.csv

Then open:
http://localhost/your-folder-name/

Notes:
- Left/Right arrow keys move between slides.
- The CSV buttons try to open the file in a new tab and also trigger download.
- Jupyter file button is still a placeholder because no .ipynb file was included yet.
